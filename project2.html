LED Message Display Box

Electronics • Soldering • Programming

Goal

The goal of this project was to use KiCAD to design a small circuit board and then solder it together to make a small device that blinks out a message. This project involved circuit design, component selection, precise soldering of surface-mount and through-hole components, and programming the display sequence. The compact design required careful attention to component placement and wire routing within size constraints.

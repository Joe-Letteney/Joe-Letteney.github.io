Automatic Pipetter For 96 Well PCR

Goal

The goal of this project was to make a device with three degrees of freedom that would solve some real world problems. Our group decided to design an automatic pipetter that would be able to dispense liquid quickly and accurately into a 96 well PCR plate. This process is usually done by hand and is long and often tedious. This device is an efficient and cheap solution to this problem.

CAD Design

To achieve the features described above, I modeled a small car in OnShape. One of the major difficulties in this project was accounting for any unevenness in the floor, so I opted for the maximum allowed length, which was one foot. We then 3D printed all of the brackets and the wheels and laser cut our base.

Controller Code

To achieve the velocity profile that we determined above, our team wrote a PID controller. This allowed us to achieve a smooth velocity profile, which is essential for this problem because dramatic shifts in the elevation profile translate to large accelerations, which would cause the bar to fall over.

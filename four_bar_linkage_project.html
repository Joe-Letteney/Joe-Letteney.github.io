Four-Bar Linkage Motion Analysis

SolidWorks • Kinematics • Motion Simulation

Goal

The goal of this project was to become familiar with SolidWorks' motion analysis feature. This included making sure that the parts are properly constrained so that the system has the correct degrees of freedom and then using the results to analyze the motion of the system.

Constraints

A key part of this project was assigning the proper constraints to each joint so that the Grübler count results in the correct degrees of freedom for the system. This process involves making sure that you have enough constraints to achieve the desired motion but that you also have no redundant constraints.

Motion Analysis

Using SolidWorks' motion analysis I was able to analyze the system with several different kinds of motion and trace the paths of different points in the system.

Results Analysis

The final part of this project was to use the motion analysis tool to determine the following results:

  1. The maximum value of the motor torque: Looking at the chart above we can see that the maximum value of the torque is approximately 6 pound force per inch.
  2. The angle of the crank for which the motor torque is maximum in each cycle: Plotting the torque and the angle of displacement at the same time we can see that the greatest torque occurs when the crank is 180 degrees from its starting position.
  3. The maximum vertical force exerted on Bearing and New Bearing: As we can see from the charts below the maximum vertical force exerted on the Bearing is around 0.07 pounds and 0.05 on the New bearing.
  4. The minimum motor torque required to move the Crank at a constant angular speed of 600rpm: As we can see from the plot below is around 537 Watts.
  5. Explain the meaning of the negative torque values and how they are related to the motion of the mechanism: The motor is attempting to move the crank at a constant angular velocity but at certain points gravity is attempting to accelerate the crank which means that in order to maintain the constant speed the motor must work against the crank.

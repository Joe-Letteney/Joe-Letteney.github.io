Bagatelle Arcade Game

Mechanical Design • Fabrication • Game Design

Goal

The goal of this project was to design and manufacture a small pinball machine using a variety of machining techniques as part of a week long hackathon style project.

Design

The first step in this project was to model the whole pinball machine in OnShape.

Manufacturing

When I had fully modeled the bagatelle I began manufacturing the individual parts. This involved using a variety of manufacturing techniques including:

  • Water jetter and sheet metal bending for the stand
  • Laser cutter for the front and back plate
  • Manual mill for the holes
  • CNC mill for the spineret
  • Manual mill for the knob
  • 3D printing for the gears
  • Aluminum casting for the name plate
